# Mico - The 10x Slacker
Mico is a funny and sarcastic slack bot that is like an annoying friend/coworker!

Check it out in the Hackclub Slack by mentioning @mico to talk to it! (doesn't work in DMs)

## Features
- Fetch project stats
- fetch user stats
- get shop items
- look at achievements
- see devlogs

## Self Hosting
To self host Mico, you need the following keys:
AI_BASE_URL=""
AI_API_KEY=""
AI_MODEL_ID=""
SLACK_BOT_TOKEN=""
SLACK_SIGNING_SECRET=""

FLAVORTOWN_API_KEY=""

You can get the AI keys from your AI provider, and the Slack keys from your Slack app settings. The FLAVORTOWN_API_KEY is optional and can be obtained from https://flavortown.hackclub.com.
